В предыдущим уроке вы узнали о том, как внешний стилевой файл, а также о основном синтаксисе.

В этом уроке вы узнаете много интересной информации для форматирования текста на CSS:

    Для начала расссмотрим параметры форматирования текста:

  • COLOR - цвет текста
  • TEXT-ALIGN - положение текста (слева, центр, справа)
  • TEXT-DECORATION - оформление текста
  • TEXT-TRANSFORM - преобразование текста
  • TEXT-INDENT - красная строка
  • DIRECTION - направление текста
  • LINE-HEIGHT - межстрочный интервал
  • LETTER-SPACING - межсимвольное расстояние
  • WORD-SPACING - расстояние между словами
    У color могут быть следующие значения:
  • red - название цвета
  • rgb(50,150,0) - код RGB
  • #66e0d6 - шестнадцатеричный код
    У text-align могут быть следующие значения:
  • center - выравнивание по центру
  • left - выравнивание по левому краю
  • right - выравнивание по правому краю
  • justify - выравнивание по ширине
    У text-decoration могут быть следующие значения:
  • overline - подчёркивание сверху
  • line-through - перечёркивание
  • underline - подчёркивание
  • blink - мигание текста
  • none - без оформления
    У text-transform могут быть следующие значения:
  • uppercase - верхний регистр
  • lowercase - нижний регистр
  • capitalize - каждое слово с заглавной буквы
  • none - без преобразования
    У direction могут быть следующие значения:
  • ltr - направление текста слева-направо
  • rtl - направление текста справа-налево-
    У line-height могут быть следующие значения:
  • line-height: normal - по умолчанию (авто)
  • line-height: 4 - множетель, который умножается на стандарт
  • line-height: 10px - в пикселях
  • line-height: 5% - в процентах
    У letter-spacing могут быть следующие значения:
  • letter-spacing: 2px - отступ между символами
  • letter-spacing: 10px - отступ между символами
  • letter-spacing: -1px - отступ между символами
    У word-spacing могут быть следующие значения:
  • word-spacing: 2px - отступ между словами
  • word-spacing: 10px - отступ между словами
  • word-spacing: -1px - отступ между словами
Сейчас рассмотрим параметры тэг{ параметр 1; параметр 2; параметр 3; } К примеру: body{ background: black; - фон будет чёрным color: green; - текст будет зелёным }

А для добавления стиля к определённой части текста без добавления внешнего файла стиля (удобно, если нужно совсем немнрго изменить стиль страницы) нужно следующее: (

В первом варианте (по отношении к конкретному тэгу в конкретной части текста страницы)

К примеру использую тэг p

test

<p style="color: green; background: black;">test<br>test</p>

Теперь стоит описать использоание классов. Классы обозначаются в тексте как class="test" а в стилевой части как .test

Во вотором варианте будет использоваться тип вложения, который заключается в том, что стилевая часть записывается отдельно от конкретного

К примеру использую тэг p

test
test

<style type="text/css"> p{ color: white; background: blue; } </style>
А теперь рассмотрим способ использования внешнего файла стиля.

Для начала необходимо создать сам стилевой файл, а для этого всего лишь необхоидмо создать файл с расширением .css

Теперь перейдём к связыванию стилевого файла и страницы web-сайта.

После этого будет необходимо внутри тэга HEAD вписать следующий код:

<link rel="stylesheet" type="text/css" href="st.css">

Внутри параметра href впишите имя файла стиля с соответственно его расширением
