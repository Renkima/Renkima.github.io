Для изменения содержимого сайта можно использовать следующие тэги (всё находится внутри тэга body:
table - тэг таблицы.
tr - тэг строки талбицы.
th - тэг ячейки таблицы.
border - параметр тэга table нужный для определения величины обводки таблицы (если авно нулю или отсутствует, то границы будут невидимы.
rowspan - параметр тэга th позволяющая расширить ячейку горизонтально.
colspan - параметр тэга th позволяющая расширить ячейку вертикально.

Вот пример таблицы:

Просто таблица (с видимой границой)
Ячейка 1 Ячейка 2
Ячейка 3 Ячейка 4 Ячейка 5
<table border="1"> <tr> <th>Ячейка 1</th> <th>Ячейка 2</th> </tr> <tr> <td>Ячейка 3</td> <td>Ячейка 4</td> <td>Ячейка 5</td> </tr> </table>
Таблица с параметрами:
Ячейка 1 Ячейка 2
Ячейка 3 Ячейка 4 Ячейка 5
<table border="1"> <tr> <th rowspan="2">Ячейка 1</th> <th colspan="3">Ячейка 2</th> </tr> <tr> <td>Ячейка 3</td> <td>Ячейка 4</td> <td>Ячейка 5</td> </tr> </table>
