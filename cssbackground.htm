  • Использование фонов

В этом уроке вы узнаете много интересной информации для работы с фоном на CSS:

    Для начала расссмотрим параметры фона:

  • background-color - цвет фона
  • background-image - изображение фона
  • background-repeat - повторение фона
  • background-position - позиционирование фона
  • background-attachment - прокрутка фона
    У background-color могут быть следующие значения:
  • red - название цвета
  • transparent - прозрачный
  • rgb(50,150,0) - код RGB
  • #66e0d6 - шестнадцатеричный код
    У background-image могут быть следующие значения:
  • url(http://kpfu.ru/docs/F85053985095/img1553513517.jpg) - картинка с интернета
  • url(i/bumagafon.png) - картинка локально
  • none - без фона
    У background-image могут быть следующие значения:
  • background-repeat: repeat-x; - повторение только горизонтально





















  • background-repeat: repeat-y; - повторение только вертикально




















  • background-repeat: no-repeat; - без повторения




















    У background-position могут быть следующие значения:
  • background-position: 40px 7px; - координаты





  • background-position: left top; - по левому и нижнему краю





  • background-position: left center; - по центру и по левому краю





  • background-position: left bottom; - по левому и верхнему





  • background-position: left top; - по левому и нижнему краю





  • background-position: right top; - по правому и нижнему краю





  • background-position: right center; - по центру и по правому краю





  • background-position: right bottom; - по правому и верхнему





  • background-position: right top; - по правому и нижнему краю





  • background-position: right top; - по правому и нижнему краю





  • background-position: right center; - по центру по вертикали и горизонтали





  • background-position: right bottom; - по центру и верхнему краю





  • background-position: right top; - по центру и нижнему краю





    У background-attachment могут быть следующие значения:
  • background-attachment: scroll; - прокрутка вместе с фоном (обычная)





















  • background-attachment: fixed - фон фиксирован
