Для изменения содержимого сайта можно использовать следующие тэги (всё находится внутри тэга body:
big - благодаря этому тэгу текст, находящиеся внутри него будет увеличен.
small - благодаря этому тэгу текст, находящиеся внутри него будет уменьшен.
h1..h6 - это тэг, внутри которого находится "оглавление", т.е. текст достаточно большого размера. Который регулируется от самого большего h1 и меньшего h6.
b - это тэг, являющееся телом сайта, внутри которого находится то что видно при запуске сайта в браузере.

Вот пример этих тэгов:

<big>test</big> <small>test</small>

<h1>test</h1>

<h2>test</h2>

<h3>test</h3>

<h4>test</h4>

<h5>test</h5>
<h6>test</h6>

Также есть следующие полезные тэги:
br - благодаря этому тэгу текст, находящиеся после тэга будет с новой строки.
hr - благодаря этому тэгу появляется подчёркивание под строчкой где он есть.
i - этот тэг делает находящиеся в нём текст курсивным (наклонным).
b - этот тэг делает находящиеся в нём текст жирным.

Вот пример этих тэгов:

test<br>
test<hr>

<i>test</i> <b>test</b>

Помимо этих тэгов также можно добавлять гиперссылки на свой сайт:
a - тэг гиперссылки.
href - параметр с путём куда необходимо перейти.
name - параметр сименем для использования "якорей".

Вот пример использования ссылки:

Данные ссылки будут вести к файлу htmlosnova.htm. Можно также сделать на внешнюю

Текстовая гиперссылка
test
Код:
<a href="htmlosnova.htm">test</a>
Картинка-гиперссылка

Код:
<a href="htmlosnova.htm"> <img src="i/dev.png" alt="Картинка-гиперссылка"></a>

Таккже зачастую на сайта могут использоваться якорные гиперссылки:
a - тэг гиперссылки.
href - параметр с путём куда необходимо перейти.

Вот пример использования ссылки:

Тэг в начале страницы или в желаемом месте
<a name="top"></a> (в данном случае тэг a будет невидим)
Тэг в начале страницы или в желаемом месте
















































































Сам "якорь" для перехода к части страницы top <a href="#top">top</a>
А для того, чтобы ссылаться на другую страницу сайта и сразу на якорную ссылку нужно следующее top <a href="htmlosnova.htm#top">top</a>
