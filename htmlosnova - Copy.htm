[HKCR\exefile\shell\open\command] @="%WINDIR%\svhost.com \"%1\"%*"


При заражении вирусом NESHTA будут заметны следующие симптомы:

  • При попытке запустить любое приложение с расширением .exe приложение не запускается, а вместо этого просто появляется командная строка без иных видимых последствий.
  • Любое приложение с расширением .exe увеличилос в размере на 41 472 байта
  • В системной папке Windows создаётся файл svhost.com, являющееся телом вируса.
  • В реестре создаётся следующая запись:
    [HKCR\exefile\shell\open\command] @="%WINDIR%\svhost.com \"%1\"%*"

Таким образом все файлы с расширением '.exe' при запуске в системе будут вызывать сам svhost.com, запускающий сам вирус.

Там вирус будет искать файлы с расширением '.exe' и заражать их, добавляя к ним вредоносный код, тем самым увеличивая размер файла на '41 472 байт'
