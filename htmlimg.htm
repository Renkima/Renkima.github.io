Использование изображеинй на сайте может пригодиться многим, а тэги для изображения следующие:
img - для добавления изображения.
src - параметр, позволяющий указать путь к изображению.
alt - параметр, выводящий текст при наведении на изображеине.

Для добавления фавикона (иконки страницы сайта) необходимо всего лишь вписать внутри HEAD

<link rel="icon" type="image/vhd.microsoft.icon" href="i/Info.ico"> При этом href указывает на путь к файлу иконки (формат только .ico)

Теперь перейдём к добавлению изображения на страницу

<img src="i/INF.png" height="auto" width="40%" title="При наведении виден этот текст"> Если выставить параметр height или width на 'auto', то изображение будет сохранять соотношение сторон.
<img src="i/Help.png" height="auto" width="40%" title="При наведении виден этот текст">
