  • HTML
  • Создание сайта




Для создания своего первого сайта для начала необходимо создать сам файл, а сделать это можно следующим образом: создать текстовый файл, а после переименовать его в 'название'.html или 'название'.htm (вместо 'название' впишите желаемое название). После этого требуется лишь открыть сам фйл при помощи практически любого текстового редактора (сойдёт даже стандартный блокнот). Однако для упрощения редактирования сайта желательно использовать редакторы с подсветкой синтаксиса, к таким редакторам относятся:

  • Notepad++
  • Brackets
  • Sublime Text

Теперь приступим к первичному редактирванию сайта. Сайт имеет по себе следующую структуру

<html> <head> <title></title> </head> <body> </body> </html>

Теперь разберём структуру сайта

html - это основа будущего сайта. Внутри тэга html будет находиться остальное содержимое.
head - это так сказать 'голова' сайта. Внутри неё находится тэг title и тэги связи с внешними стилями и фавиконом.
title - это тэг, внутри которого находится название сайта, которое отображается в названии вкладки браузера.
body - это тэг, являющееся телом сайта, внутри которого находится то что видно при запуске сайта в браузере.

Вот пример сайта (код)

<html> <head> <title>Пример сайта</title> </head> <body> Hello World ! </body> </html>
